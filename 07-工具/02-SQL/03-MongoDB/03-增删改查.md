# 2. 集合操作

## 2.1 增加

```shell

```

* capped: 是否为固定集合
* size: 上限大小

## 2.2 删除


# 3. 数据操作

## 3.1 增加

```shell
db.spider.insert({name: "ww", age: 11})

data = {{name: "hh"}}
data.age = 22
db.spider.insert(data)
```

* 注意key可以不加引号, 但是如果有一个加引号, 那么全部都要加引号.

## 3.2 删除

```shell
db.spider.remove({"name": "ww"})  # 匹配删除
db.spider.remove({})              # 全部删除
```
